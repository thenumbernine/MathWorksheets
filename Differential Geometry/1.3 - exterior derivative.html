Wedge product linearity:
$(sa) \wedge b = a \wedge (sb) = s(a \wedge b)$ for scalar / 0-form s, p-form a, q-form b

Wedge product antisymmetry: $a \wedge b = -b \wedge a$

Wedge product of coordinate basis one-forms:
$dx^a \wedge dx^b = dx^a \otimes dx^b - dx^b \otimes dx^a$

Wedge product coordinate basis antisymmetry:
$dx^a \wedge dx^b = dx^a \otimes dx^b - dx^b \otimes dx^a$
$ - (dx^b \otimes dx^a - dx^a \otimes dx^b)$
$= -dx^b \wedge dx^a$

Wedge product of non-coordinate basis:
$e^a \wedge e^b = {e^a}_\bar{a} dx^a \wedge {e^b}_\bar{b} dx^b$
$= {e^a}_\bar{a} {e^b}_\bar{b} dx^a \wedge dx^b$
$= -{e^a}_\bar{a} {e^b}_\bar{b} dx^b \wedge dx^a$
$= -e^b \wedge e^a$

wedge product of one-forms:
$a \wedge b = a_u e^u \wedge b_v e^v$
$ = a_u b_v e^u \wedge e^v$
$ = a_u b_v (e^u \otimes e^v - e^v \otimes e^u)$
$ = (a_u b_v - b_u a_v) e^u \otimes e^v$
$ = a \otimes b - b \otimes a$

Wedge product of p-form basis: $ e^{u_1} \wedge ... \wedge e^{u_p} = p! \cdot e^{[u_1} \otimes ... \otimes e^{u_p]}$

If a rank-p covariant tensor has a completely antisymmetric basis then it is said to be a p-form.

Wedge product antisymmetry for p-form $a$ and q-form $b$:
$ a \wedge b $
$ = (a_{u_1 ... u_p} e^{u_1} \wedge ... \wedge e^{u_p}) \wedge (b_{v_1 ... v_q} e^{v_1} \wedge ... \wedge e^{v_q}) $
if any of $ u_1 ... u_p, v_q ... v_q $ are not unique then the result is zero (courtesy of $ e^u \wedge e^u = 0 $ ).
Otherwise:
$ = a_{u_1 ... u_p} b_{v_1 ... v_q} (e^{u_1} \wedge ... \wedge e^{u_p}) \wedge (e^{v_1} \wedge ... \wedge e^{v_q}) $
Rotate the first $ e^{v_1} $ to the far left side using $ e^u \wedge e^v = -e^v \wedge e^u $. Doing so requires $p$ applications of the rule.
$ = (-1)^p a_{u_1 ... u_p} b_{v_1 ... v_q} e^{v_1} \wedge (e^{u_1} \wedge ... \wedge e^{u_p}) \wedge e^{v_2} \wedge ... \wedge e^{v_q} $
Continue to do the same for the rest of the $e^{v_k}$'s, doung so a number of $q$ times:
$ = (-1)^{pq} a_{u_1 ... u_p} b_{v_1 ... v_q} (e^{v_1} \wedge ... \wedge e^{v_q}) \wedge (e^{u_1} \wedge ... \wedge e^{u_p}) $
$ = (-1)^{pq} (b_{v_1 ... v_q} e^{v_1} \wedge ... \wedge e^{v_q}) \wedge (a_{u_1 ... u_p} e^{u_1} \wedge ... \wedge e^{u_p}) $
$ = (-1)^{pq} b \wedge a $

Wedge products and forms in terms of index notation
For p-form A:
$\frac{1}{p!} A_{a_1 ... a_p} \delta^{a_1 ... a_p b_1 ... b_p}_{c_1 ... c_p c_{p+1} ... c_{p+q}}$ $= \underset{a_1 < a_2 < ... < a_p}{\Sigma} A_{a_1 ... a_p} \delta^{a_1 ... a_p b_1 ... b_p}_{c_1 ... ... c_{p+q}}$ $=A_{|a_1 ... a_p|} \delta^{a_1 ... a_p b_1 ... b_p}_{c_1 ...... c_{p+q}}$
$\delta^{a_1 ... a_p b_1 ... b_q}_{c_1 ... ... c_{p+q}} \delta^{u_1 ... u_q}_{|b_1 ... b_q|} = \delta^{a_1 ... a_p u_1 ... u_q}_{c_1 ... ... c_{p+q}}$
For p-form a, q-form b:
$(a \wedge b)_{d_1 ... d_{p+q}} = \delta^{u_1 ... u_p v_1 ... v_q}_{d_1 ... d_p d_{p+1} ... d_{p+q}} a_{|u_1 ... u_p|} b_{|v_1 ... v_q|}$
For rank-p vector U and rank-q vector V:
$(U \wedge V)^{d_1 ... d_{p+q}} = \delta^{d_1 ... d_p d_{p+1} ... d_{p+q}}_{u_1 ... u_p v_1 ... v_q} U^{|u_1 ... u_p|} V^{|v_1 ... v_q|}$
For p-form a, q-form b, r-form c:
$[( a \wedge b ) \wedge {c} ]_{a_1 ... a_{p+q+r}}$ $= \delta^{d_1 ... d_p u_1 ... u_q v_1 ... v_r}_{e_1 ... ... ... e_{p+q+r}} a_{|d_1 ... d_p|} b_{|u_1 ... u_q|} c_{|v_1 ... v_r|}$ $= [a \wedge (b \wedge {c})]_{e_1 ... e_{p+q+r}}$
For vector $u_i$:
$(u_1 \wedge ... \wedge u_p)^{a_1 ... a_p}$ $= \delta^{a_1 ... a_p}_{u_1 ... u_p} (u_1)^{u_1} ... (u_p)^{u_p}$ $= p! {u_1}^{[a_1} ... {u_p}^{a_p]}$ $= \delta^{a_1 ... a_p}_{1 ... p} det({u_u}^d)$

Exterior derivative of one-form in a coordinate basis:
$dw = d(w_b {dx}^b)$
$ = dw_b \wedge {dx}^b + w_b d^2{x}^b$
$ = dw_b \wedge {dx}^b$

Exterior derivative of a p-form:
$d w = d (w_{i_1 ... i_p} e^{i_1} \wedge ... \wedge e^{i_p})$
$= d (w_{i_1 ... i_p}) \wedge e^{i_1} \wedge ... \wedge e^{i_p} + (-1)^{j-1} w_{i_1 ... i_p} e^{i_1} \wedge ... \wedge d(e^{i_j}) \wedge ... \wedge e^{i_p} $

...in a coordinate basis, where $e^\bar{a} = dx^\bar{a}$ and $d(dx^\bar{a}) = 0$:
$= \partial_j (w_{i_1 ... i_p}) dx^j \wedge dx^{i_1} \wedge ... \wedge dx^{i_p}$

Exterior derivative of a non-coordinate basis one-form:
(Treating the one-form basis as a linear combination of a coordinate basis)
$d e^v = d ({e^v}_\bar{b} dx^\bar{b})$
$= d ({e^v}_\bar{b}) \wedge dx^\bar{b} + {e^v}_\bar{b} d(dx^\bar{b})$
$= \partial_\bar{a} ({e^v}_\bar{b}) dx^\bar{a} \wedge dx^\bar{b}$
$= \partial_\bar{a} ({e^v}_\bar{b}) dx^\bar{a} \wedge dx^\bar{b}$
$= e_a ({e^v}_\bar{b}) {e_b}^\bar{b} e^a \wedge e^b$
$= e_a ({e^v}_\bar{b}) {e_b}^\bar{b} (e^a \otimes e^b - e^b \otimes e^a)$
$= (e_a ({e^v}_\bar{b}) {e_b}^\bar{b} - e_b ({e^v}_\bar{b}) {e_a}^\bar{b} ) e^a \otimes e^b$
$= -(e_a ({e_b}^\bar{b}) - e_b ({e_a}^\bar{b})) {e^v}_\bar{b} e^a \otimes e^b$
$= -{c_{ab}}^u e^a \otimes e^b$
$= -\frac{1}{2}{c_{ab}}^u e^a \wedge e^b$
which looks like the Maurer-Cartan equation.

Exterior derivative of a p-form in a non-coordinate basis:
$dw = e_k (w_{i_1 ... i_p}) e^k \wedge e^{i_1} \wedge ... \wedge e^{i_p} + (-1)^{j-1} w_{i_1 ... i_p} e^{i_1} \wedge ... \wedge d(e^{i_j}) \wedge ... \wedge e^{i_p}$
$ = e_k (w_{i_1 ... i_p}) e^k \wedge e^{i_1} \wedge ... \wedge e^{i_p} + (-1)^{j-1} w_{i_1 ... i_p} e^{i_1} \wedge ... \wedge (-\frac{1}{2} {c_{ab}}^{i_j} e^a \wedge e^b) \wedge ... \wedge e^{i_p} $
$ = e_k (w_{i_1 ... i_p}) e^k \wedge e^{i_1} \wedge ... \wedge e^{i_p} - \frac{1}{2} {c_{ab}}^{i_j} w_{i_1 ... i_p} e^a \wedge e^{i_1} \wedge ... \wedge e^b \wedge ... \wedge e^{i_p} $
$ = ( e_k (w_{i_1 ... i_p}) - \frac{1}{2} {c_{k i_j}}^b w_{i_1 ... i_{j-1} b i_{j+1} ... i_p} ) e^k \wedge e^{i_1} \wedge ... \wedge e^{i_p} $
