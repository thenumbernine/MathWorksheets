covariant derivative in one-form basis:

Riemann curvature in coordinate-free notation:
(Thanks to Bonbien & Nail for correcting this)
$R(x,y)z$
$= \nabla_x \nabla_y z - \nabla_y \nabla_x z - \nabla_{[x,y]} z$
$= \nabla_{x^a e_a} \nabla_{y^b e_b} (z^c e_c) - \nabla_{y^a e_a} \nabla_{x^b e_b} (z^c e_c) - \nabla_{[x,y]^a e_a} (z^c e_c)$
$= \nabla_{x^a e_a} \nabla_{y^b e_b} (z^c e_c) - \nabla_{y^a e_a} \nabla_{x^b e_b} (z^c e_c) - [x,y]^d \nabla_d (z^c e_c)$
$= x^a \nabla_a (y^b \nabla_b (z^c e_c)) - y^a \nabla_a (x^b \nabla_b (z^c e_c)) - (x^a e_a (y^d) - y^a e_a (x^d) + x^a y^b {c_{ab}}^d) \nabla_d (z^c e_c)$
$= x^a \nabla_a y^b \nabla_b (z^c e_c) + x^a y^b \nabla_a \nabla_b (z^c e_c) - y^a \nabla_a x^b \nabla_b (z^c e_c) - y^a x^b \nabla_a \nabla_b (z^c e_c) - (x^a e_a (y^d) - y^a e_a (x^d) + x^a y^b {c_{ab}}^d) \nabla_d (z^c e_c)$
$= (x^a e_a (y^d) - y^a e_a (x^d)) \nabla_d (z^c e_c) + (x^a y^b - y^a x^b) \nabla_a \nabla_b (z^c e_c) - (x^a e_a (y^d) - y^a e_a (x^d) + x^a y^b {c_{ab}}^d) \nabla_d (z^c e_c)$
$= (x^a y^b - y^a x^b) \nabla_a \nabla_b (z^c e_c) - x^a y^b {c_{ab}}^d \nabla_d (z^c e_c)$
$= (x^a y^b - y^a x^b) \nabla_a (e_b (z^c) e_c + z^c \nabla_b e_c) - x^a y^b {c_{ab}}^d (e_d (z^c) e_c + z^c \nabla_d e_c)$
$= (x^a y^b - y^a x^b) \nabla_a (e_b (z^c) e_c + z^c {\Gamma^e}_{bc} e_e) - x^a y^b {c_{ab}}^d (e_d (z^c) e_c + z^c {\Gamma^e}_{dc} e_e)$
$= (x^a y^b - y^a x^b) ( \nabla_a (e_b (z^c)) e_c + e_b (z^c) \nabla_a e_c + \nabla_a z^c {\Gamma^e}_{bc} e_e + z^c \nabla_a {\Gamma^e}_{bc} e_e + z^c {\Gamma^e}_{bc} \nabla_a e_e ) - x^a y^b {c_{ab}}^d (e_d (z^c) e_c + z^c {\Gamma^e}_{dc} e_e)$
$= (x^a y^b - y^a x^b) ( e_a (e_b (z^c)) e_c + e_b (z^c) {\Gamma^e}_{ac} e_e + e_a (z^c) {\Gamma^e}_{bc} e_e + z^c e_a ({\Gamma^e}_{bc}) e_e + z^c {\Gamma^e}_{bc} {\Gamma^d}_{ae} e_d ) - x^a y^b {c_{ab}}^d (e_d (z^c) e_c + z^c {\Gamma^e}_{dc} e_e)$
$= (x^a y^b - y^a x^b) ( e_a (e_b (z^c)) e_c + z^c e_a ({\Gamma^e}_{bc}) e_e + z^c {\Gamma^e}_{bc} {\Gamma^d}_{ae} e_d ) - x^a y^b {c_{ab}}^d (e_d (z^c) e_c + z^c {\Gamma^e}_{dc} e_e)$
$= x^a y^b ( e_a (e_b (z^c)) - e_b (e_a (z^c)) ) e_c + x^a y^b z^c ( + e_a ({\Gamma^e}_{bc}) - e_b ({\Gamma^e}_{ac}) + {\Gamma^d}_{bc} {\Gamma^e}_{ad} - {\Gamma^d}_{ac} {\Gamma^e}_{bd} - {c_{ab}}^d {\Gamma^e}_{dc} ) e_e - x^a y^b {c_{ab}}^d e_d (z^c) e_c $
$= z^b x^c y^d ( e_c ({\Gamma^a}_{db}) - e_d ({\Gamma^a}_{cb}) + {\Gamma^a}_{ce} {\Gamma^e}_{db} - {\Gamma^a}_{de} {\Gamma^e}_{cb} - {\Gamma^a}_{eb} {c_{cd}}^e ) e_a + x^a y^b ([e_a, e_b] (z^c)) e_c - x^a y^b {c_{ab}}^d e_d (z^c) e_c $
$= z^b x^c y^d ( e_c ({\Gamma^a}_{db}) - e_d ({\Gamma^a}_{cb}) + {\Gamma^a}_{ce} {\Gamma^e}_{db} - {\Gamma^a}_{de} {\Gamma^e}_{cb} - {\Gamma^a}_{eb} {c_{cd}}^e ) e_a + x^a y^b {c_{ab}}^d e_d (z^c) e_c - x^a y^b {c_{ab}}^d e_d (z^c) e_c $
$= z^b x^c y^d ( e_c ({\Gamma^a}_{db}) - e_d ({\Gamma^a}_{cb}) + {\Gamma^a}_{ce} {\Gamma^e}_{db} - {\Gamma^a}_{de} {\Gamma^e}_{cb} - {\Gamma^a}_{eb} {c_{cd}}^e ) e_a $
$= {R^a}_{bcd} z^b x^c y^d e_a$
