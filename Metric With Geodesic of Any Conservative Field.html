Let $x^I$ be the Minkowski coordinates
Let $x^u$ be the curved coordinates
I'll also use $x^\bar{a}$ for flat space coordinates and $x^\tilde{a}$ for curved space coordintes

Let $x^\bar{0} = \phi(x^u)$, $x^\bar{i} = x^\tilde{i}$

The basis is defined as:
${e_u}^\bar{0} = \frac{\partial x^\bar{0}}{\partial x^u} = {x^\bar{0}}_{,u} = \phi_{,u}$
${e_u}^\bar{i} = \frac{\partial x^\bar{i}}{\partial x^u} = {x^\bar{i}}_{,u} = \delta^i_u$

The inverse basis is defined as:
${e^\tilde{0}}_\bar{0} = (\phi_{,\tilde{0}})^{-1}$
${e^\tilde{0}}_\bar{i} = -(\phi_{,\tilde{i}}) (\phi_{,\tilde{0}})^{-1}$
${e^u}_\bar{i} = \delta^u_i$

Metric:
$g_{uv} = {e_u}^I {e_v}^J \eta_{IJ}$ $= -{e_u}^\bar{0} {e_v}^\bar{0} + {e_u}^\bar{i} {e_v}^\bar{j} \eta_{\bar{i}\bar{j}}$ $= -\phi_{,u} \phi_{,v} + \delta^i_u \delta^j_v \eta_{ij}$
$g_{\tilde{0}\tilde{0}} = -(\phi_{,\tilde{0}})^2$
$g_{\tilde{0}\tilde{i}} = -\phi_{,\tilde{0}} \phi_{,\tilde{i}}$
$g_{\tilde{i}\tilde{j}} = -\phi_{,\tilde{i}} \phi_{,\tilde{j}} + \delta_{ij}$

Metric inverse:
$g^{\tilde{0}\tilde{0}} = (\phi_{,\tilde{0}})^{-2} (-1 + \Sigma_{k=1}^n (\phi_{,\tilde{k}})^2)$
$g^{\tilde{i}\tilde{0}} = -(\phi_{,\tilde{i}}) (\phi_{,\tilde{0}})^{-1}$
$g^{\tilde{i}\tilde{j}} = \delta^{ij}$

Metric partial:
$g_{uv,w} = -(\phi_{,u} \phi_{,v})_{,w}$ $=-\phi_{,uw} \phi_{,v} - \phi_{,u} \phi_{,vw}$

Connection
$\Gamma_{uvw} = \frac{1}{2} (g_{uv,w} + g_{uw,v} - g_{vw,u})$
$= \frac{1}{2} ( -\phi_{,uw} \phi_{,v} - \phi_{,u} \phi_{,vw} -\phi_{,uv} \phi_{,w} - \phi_{,u} \phi_{,wv} +\phi_{,vu} \phi_{,w} + \phi_{,v} \phi_{,wu} )$ $= - \phi_{,u} \phi_{,vw}$

${\Gamma^u}_{vw} = g^{ua} \Gamma_{avw}$ $ = -g^{ua} \phi_{,a} \phi_{,vw}$
${\Gamma^\tilde{0}}_{vw} = -g^{\tilde{0}\tilde{0}} \phi_{,\tilde{0}} \phi_{,vw} - g^{\tilde{0}\tilde{i}} \phi_{,\tilde{i}} \phi_{,vw}$ $ = ( -(\phi_{,\tilde{0}})^{-2} (-1 + \Sigma_{k=1}^n (\phi_{,\tilde{k}})^2) \phi_{,\tilde{0}} + \phi_{,\tilde{i}} (\phi_{,\tilde{0}})^{-1} \phi_{,\tilde{i}} ) \phi_{,vw}$
$ = ( 1 - \Sigma_{k=1}^n (\phi_{,\tilde{k}})^2 + (\phi_{,\tilde{i}})^{2} ) (\phi_{,\tilde{0}})^{-1} \phi_{,vw}$

${\Gamma^\tilde{i}}_{vw} = g^{\tilde{i}\tilde{0}} \Gamma_{\tilde{0}vw} + g^{\tilde{i}\tilde{j}} \Gamma_{\tilde{j}vw}$
$ = -g^{\tilde{i}\tilde{0}} \phi_{,\tilde{0}} \phi_{,vw} - g^{\tilde{i}\tilde{j}} \phi_{,\tilde{j}} \phi_{,vw}$ $ = (\phi_{,\tilde{i}}) (\phi_{,\tilde{0}})^{-1} \phi_{,\tilde{0}} \phi_{,vw} - \delta^{ij} \phi_{,\tilde{j}} \phi_{,vw}$
$ = \phi_{,\tilde{i}} \phi_{,vw} - \phi_{,\tilde{i}} \phi_{,vw}$
$ = 0$

So this gives the opposite of what I wanted: A controllable connection in the new dimension for vectors at rest in the subspace, and no connection in the new dimension for any vectors - especially those at rest in the new dimension.
