Indexes: $\alpha-\omega$, $a-h$ and $o-z$ = spacetime curved, $i-n$ = spatial curved, $A-Z$ and $\hat a-\hat z$ = spacetime flat (Minkowski)
Affine covariant and Lorentz covariant connection notation standards:
Affine connection: $ {\Gamma^a}_{bc} $ uses the first two terms to denote the linear transformation of the source tensor; and uses the third index to denote which derivative this is in regards to. Probably due to the old comma derivative convention when the derivative index was placed last.
Lorentz/spin connection: $ {\omega_{aI}}^J $ uses the last two indexes to denote the linear transformation of source tensor; and the first to denote which derivative this is in regards to. Probably due to the change in standards from using comma derivatives to writing prefix $ \partial $ symbols, where the derivative index would be seen first.
The two tensors are distinctly different, though some sources use the same symbol and rely on the use of coordinate vs Lorentz indexes to distinguish which tensor is used. This causes the representation of the connection transformed from one coordinate system to another to be ambiguous.
To get around this, I'll choose separate symbols for the connections of coordinates and the connections of Lorentz indexes.
A bigger fix is needed for connections:
(1) to pick one set of index ordering, rather than having a separate ordering for the affine connection.
I'll use the new standard, derivative-index-first, for all (pseudo)tensors except the traditional affine connection.
(Maybe I'll eventually use it with the affine as well. After all, it does match up with the (Walton 1967) version: $ \{ {{_a}^c}_b \} $.)
(2) to somehow denote a relationship between a particular covariant derivative's coordinate connection and Lorentz connection.
I'd like to use $ {A_{ab}}^c $ vs $ {\hat A_{a\hat b}}^\hat c $ for coordinate vs Lorentz connection tensors. This way indexes can be transformed between coordinate and Lorentz without ambiguity.
Instead I'm going to stick with tradition.

curvature on Lorentz indexes solely in terms of spin connection:
$ {{\Omega_{ab}}^I}_J V^J = D_a D_b V^I - D_b D_a V^I $
because $\nabla_a \nabla_b - \nabla_b \nabla_a$ gives us a result in terms of ${\omega_a}^{IJ}$ and cancels out all instances of ${\Gamma^a}_{bc}$, then we can equivalently define the curvature tensor solely on the Lorentz indexes
