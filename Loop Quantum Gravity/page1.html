covariant derivative of vector in index notation: $ \nabla_\alpha v^\beta = \partial_\alpha v^\beta + {\Gamma^\beta}_{\gamma\alpha} v^\gamma $

covariant derivative of one-form: $ \boldsymbol{\nabla} \textbf{w} = \nabla_\alpha \textbf{w} \otimes \boldsymbol{\omega}^\alpha = \nabla_\alpha (w_\gamma \boldsymbol{\omega}^\gamma) \otimes \boldsymbol{\omega}^\alpha = (\nabla_\alpha w_\gamma \boldsymbol{\omega}^\gamma + w_\gamma \nabla_\alpha \boldsymbol{\omega}^\gamma) \otimes \boldsymbol{\omega}^\alpha = (\partial_\alpha w_\beta - {\Gamma^\gamma}_{\beta\alpha} w_\gamma ) \boldsymbol{\omega}^\beta \otimes \boldsymbol{\omega}^\alpha $
covariant derivative of one-form in index notation: $ \nabla_\alpha w_\beta = \partial_\alpha w_\beta - {\Gamma^\gamma}_{\beta\alpha} w_\gamma $

This is where we see that in index notation $ \nabla_\gamma $ treats $ v^\alpha $ as a vector due to its raised index, while in coordinate-free notation $\nabla_\gamma $ treats $ v^\alpha $ as a scalars of the $ \textbf{e}_\alpha $ vector.

covariant derivative of metric in index notation: $ \nabla_\gamma g_{\alpha\beta} = \partial_\gamma g_{\alpha\beta} - {\Gamma^\delta}_{\alpha\gamma} g_{\delta\beta} - {\Gamma^\delta}_{\beta\gamma} g_{\alpha\delta} = \partial_\gamma g_{\alpha\beta} - \Gamma_{\beta\alpha\gamma} - \Gamma_{\alpha\beta\gamma} = \partial_\gamma g_{\alpha\beta} - \partial_\gamma g_{\alpha\beta} = 0 $

covariant vector derivative: $ \nabla_{\textbf{u}} \textbf{v} = \textbf{u} \cdot \boldsymbol{\nabla} \textbf{v} = u^\alpha \textbf{e}_\alpha \cdot \nabla_\beta (v^\gamma \textbf{e}_\gamma) \otimes \boldsymbol{\omega}^\beta = u^\alpha \textbf{e}_\alpha \cdot (\partial_\beta v^\gamma + v^\delta {\Gamma^\gamma}_{\delta\beta}) \textbf{e}_\gamma \otimes \boldsymbol{\omega}^\beta = u^\alpha (\partial_\beta v^\gamma + v^\delta {\Gamma^\gamma}_{\delta\beta}) \textbf{e}_\alpha \cdot \textbf{e}_\gamma \otimes \boldsymbol{\omega}^\beta = u^\alpha (\partial_\beta v^\gamma + v^\delta {\Gamma^\gamma}_{\delta\beta}) g_{\alpha\gamma} \boldsymbol{\omega}^\beta = u_\gamma (\partial_\beta v^\gamma + v^\alpha {\Gamma^\gamma}_{\alpha\beta}) \boldsymbol{\omega}^\beta $

Riemann curvature in terms of covariant derivative, in coordinate-free notation:
${R^\alpha}_{\beta\gamma\delta} = \langle \boldsymbol{\omega}^\alpha, (\left [ \nabla_\gamma, \nabla_\delta \right ] - \nabla_{[\gamma,\delta]}) \textbf{e}_\beta \rangle $
$=\langle \boldsymbol{\omega}^\alpha, 2 \nabla_{[\gamma} \nabla_{\delta]} \textbf{e}_\beta - \nabla_{[\gamma,\delta]} \textbf{e}_\beta \rangle $
$=\langle \boldsymbol{\omega}^\alpha, 2 \nabla_{[\gamma} ({\Gamma^\mu}_{\beta|\delta]} \textbf{e}_\mu) - \nabla_{[\gamma,\delta]} \textbf{e}_\beta \rangle $
$=\langle \boldsymbol{\omega}^\alpha, 2 \nabla_{[\gamma} {\Gamma^\nu}_{\beta|\delta]} \textbf{e}_\nu + 2 {\Gamma^\nu}_{\beta[\delta} \nabla_{\gamma]} \textbf{e}_\nu - {\Gamma^\nu}_{\beta\mu} {\Gamma^\mu}_{[\delta\gamma]} \textbf{e}_\nu \rangle $
$=2 \langle \boldsymbol{\omega}^\alpha, \textbf{e}_\nu \rangle (\partial_{[\gamma} {\Gamma^\nu}_{\beta|\delta]} + {\Gamma^\mu}_{\beta[\delta} {\Gamma^\nu}_{\mu|\gamma]} - {\Gamma^\nu}_{\beta\mu} {\Gamma^\mu}_{[\delta\gamma]}) $
$=2(\partial_{[\gamma} {\Gamma^\alpha}_{\beta|\delta]} + {\Gamma^\alpha}_{\mu[\gamma} {\Gamma^\mu}_{\beta|\delta]} - {\Gamma^\alpha}_{\beta\mu} {\Gamma^\mu}_{[\delta\gamma]}) $

Riemann curvature in terms of covariant derivative, in index notation:
$ (\left [ \nabla_\gamma, \nabla_\delta \right ] - \nabla_{[\gamma,\delta]} ) v^\alpha$
$= 2 \nabla_{[\gamma} \nabla_{\delta]} v^\alpha - \nabla_{[\gamma,\delta]} v^\alpha$
expand inner $\nabla$
$= 2 \nabla_{[\gamma} (\partial_{\delta]} v^\alpha + {\Gamma^\alpha}_{\beta|\delta]} v^\beta) - \nabla_{[\gamma, \delta]} v^\alpha$
distribute outer $\nabla$
$= 2 (\nabla_{[\gamma} \partial_{\delta]} v^\alpha + \nabla_{[\gamma} ({\Gamma^\alpha}_{\beta|\delta]} v^\beta)) - \nabla_{[\gamma,\delta]} v^\alpha$
distribute first $\nabla$
$= 2 (\partial_{[\gamma} \partial_{\delta]} v^\alpha - {\Gamma^\mu}_{[\delta\gamma]} \partial_\mu v^\alpha + {\Gamma^\alpha}_{\mu[\gamma} \partial_{\delta]} v^\mu + \nabla_{[\gamma} ({\Gamma^\alpha}_{\beta|\delta]} v^\beta)) - \nabla_{[\gamma,\delta]} v^\alpha$
cancel symmetric term, distribute second $\nabla$
$= 2 (-{\Gamma^\mu}_{[\delta\gamma]} \partial_\mu v^\alpha + {\Gamma^\alpha}_{\mu[\gamma} \partial_{\delta]} v^\mu + \partial_{[\gamma} ({\Gamma^\alpha}_{\beta|\delta]} v^\beta) + {\Gamma^\alpha}_{\mu[\gamma} {\Gamma^\mu}_{\beta|\delta]} v^\beta - {\Gamma^\mu}_{[\delta\gamma]} {\Gamma^\alpha}_{\beta\mu} v^\beta ) - \nabla_{[\gamma,\delta]} v^\alpha$
group like $\Gamma$'s, distribute first $\partial$
$= 2 ({\Gamma^\alpha}_{\mu[\gamma} \partial_{\delta]} v^\mu + \partial_{[\gamma} {\Gamma^\alpha}_{\beta|\delta]} v^\beta + {\Gamma^\alpha}_{\beta[\delta} \partial_{\gamma]} v^\beta + {\Gamma^\alpha}_{\mu[\gamma} {\Gamma^\mu}_{\beta|\delta]} v^\beta + {\Gamma^\mu}_{[\gamma\delta]} (\partial_\mu v^\alpha + {\Gamma^\alpha}_{\beta\mu} v^\beta) ) - \nabla_{[\gamma,\delta]} v^\alpha$
cancel symmetric first and third terms
$= 2 (\partial_{[\gamma} {\Gamma^\alpha}_{\beta|\delta]} v^\beta + {\Gamma^\alpha}_{\mu[\gamma} {\Gamma^\mu}_{\beta|\delta]} v^\beta + {\Gamma^\mu}_{[\gamma\delta]} (\partial_\mu v^\alpha + {\Gamma^\alpha}_{\beta\mu} v^\beta) ) - \nabla_{[\gamma,\delta]} v^\alpha$
substitute $\nabla_{[\gamma,\delta]} v^\alpha = {\Gamma^\mu}_{[\gamma\delta]} \partial_\mu v^\alpha$ and cancel
$= 2 (\partial_{[\gamma} {\Gamma^\alpha}_{\beta|\delta]} v^\beta + {\Gamma^\alpha}_{\mu[\gamma} {\Gamma^\mu}_{\beta|\delta]} v^\beta + {\Gamma^\mu}_{[\gamma\delta]} {\Gamma^\alpha}_{\beta\mu} v^\beta )$
factor $v^\beta$
$= 2 (\partial_{[\gamma} {\Gamma^\alpha}_{\beta|\delta]} + {\Gamma^\alpha}_{\mu[\gamma} {\Gamma^\mu}_{\beta|\delta]} + {\Gamma^\alpha}_{\beta\mu} {\Gamma^\mu}_{[\gamma\delta]} ) v^\beta$
Let $R = 2 (\partial_{[\gamma} {\Gamma^\alpha}_{\beta|\delta]} + {\Gamma^\alpha}_{\mu[\gamma} {\Gamma^\mu}_{\beta|\delta]} + {\Gamma^\alpha}_{\beta\mu} {\Gamma^\mu}_{[\gamma\delta]} )$
So $([\nabla_\gamma,\nabla_\delta] - \nabla_{[\gamma,\delta]}) v^\alpha = {R^\alpha}_{\beta\gamma\delta} v^\beta$

Ricci curvature: $ R_{\alpha\beta} = {R^\gamma}_{\alpha\gamma\beta} $
Gaussian curvature: $ R = {R^\alpha}_\alpha $
