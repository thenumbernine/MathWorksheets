$\nabla_u v = \nabla_{u^a e_a} (v^b e_b) = u^a \nabla_a (v^b e_b) = u^a (\partial_a v^b + {\Gamma^b}_{ca} v^c) e_b$

Riemann curvature in coordinate-free notation:
(Thanks to Bonbien & Nail for correcting this)
$R(x,y)z$
$= \nabla_x \nabla_y z - \nabla_y \nabla_x z - \nabla_{[x,y]} z$
$= \nabla_{x^a e_a} \nabla_{y^b e_b} (z^c e_c) - \nabla_{y^a e_a} \nabla_{x^b e_b} (z^c e_c) - \nabla_{[x,y]^a e_a} (z^c e_c)$
$= \nabla_{x^a e_a} \nabla_{y^b e_b} (z^c e_c) - \nabla_{y^a e_a} \nabla_{x^b e_b} (z^c e_c) - [x,y]^d \nabla_d (z^c e_c)$
$= x^a \nabla_a (y^b \nabla_b (z^c e_c)) - y^a \nabla_a (x^b \nabla_b (z^c e_c)) - (x^a e_a y^d - y^a e_a x^d + x^a y^b {c_{ab}}^d) \nabla_d (z^c e_c)$
$= x^a \nabla_a y^b \nabla_b (z^c e_c) + x^a y^b \nabla_a \nabla_b (z^c e_c) - y^a \nabla_a x^b \nabla_b (z^c e_c) - y^a x^b \nabla_a \nabla_b (z^c e_c) - (x^a e_a y^d - y^a e_a x^d + x^a y^b {c_{ab}}^d) \nabla_d (z^c e_c)$
$= (x^a e_a y^d - y^a e_a x^d) \nabla_d (z^c e_c) + (x^a y^b - y^a x^b) \nabla_a \nabla_b (z^c e_c) - (x^a e_a y^d - y^a e_a x^d + x^a y^b {c_{ab}}^d) \nabla_d (z^c e_c)$
$= (x^a y^b - y^a x^b) \nabla_a \nabla_b (z^c e_c) - x^a y^b {c_{ab}}^d \nabla_d (z^c e_c)$
$= (x^a y^b - y^a x^b) \nabla_a ((e_b z^c) e_c + z^c \nabla_b e_c) - x^a y^b {c_{ab}}^d ((e_d z^c) e_c + z^c \nabla_d e_c)$
$= (x^a y^b - y^a x^b) \nabla_a ((e_b z^c) e_c + z^c {\Gamma^e}_{cb} e_e) - x^a y^b {c_{ab}}^d ((e_d z^c) e_c + z^c {\Gamma^e}_{cd} e_e)$
$= (x^a y^b - y^a x^b) ( \nabla_a (e_b z^c) e_c + (e_b z^c) \nabla_a e_c + \nabla_a z^c {\Gamma^e}_{cb} e_e + z^c \nabla_a {\Gamma^e}_{cb} e_e + z^c {\Gamma^e}_{cb} \nabla_a e_e ) - x^a y^b {c_{ab}}^d ((e_d z^c) e_c + z^c {\Gamma^e}_{cd} e_e)$
$= (x^a y^b - y^a x^b) ( e_a (e_b z^c) e_c + (e_b z^c) {\Gamma^e}_{ca} e_e + (e_a z^c) {\Gamma^e}_{cb} e_e + z^c (e_a {\Gamma^e}_{cb}) e_e + z^c {\Gamma^e}_{cb} {\Gamma^d}_{ea} e_d ) - x^a y^b {c_{ab}}^d ((e_d z^c) e_c + z^c {\Gamma^e}_{cd} e_e)$
$= (x^a y^b - y^a x^b) ( e_a (e_b z^c) e_c + z^c (e_a {\Gamma^e}_{cb}) e_e + z^c {\Gamma^e}_{cb} {\Gamma^d}_{ea} e_d ) - x^a y^b {c_{ab}}^d ((e_d z^c) e_c + z^c {\Gamma^e}_{cd} e_e)$
$= x^a y^b ( e_a (e_b z^c) - e_b (e_a z^c) ) e_c + x^a y^b z^c ( + (e_a {\Gamma^e}_{cb}) - (e_b {\Gamma^e}_{ca}) + {\Gamma^d}_{cb} {\Gamma^e}_{da} - {\Gamma^d}_{ca} {\Gamma^e}_{db} - {c_{ab}}^d {\Gamma^e}_{cd} ) e_e - x^a y^b {c_{ab}}^d (e_d z^c) e_c $
$= z^b x^c y^d ( e_c {\Gamma^a}_{bd} - e_d {\Gamma^a}_{bc} + {\Gamma^a}_{ec} {\Gamma^e}_{bd} - {\Gamma^a}_{ed} {\Gamma^e}_{bc} - {\Gamma^a}_{be} {c_{cd}}^e ) e_a + x^a y^b ([e_a, e_b] z^c) e_c - x^a y^b {c_{ab}}^d (e_d z^c) e_c $
$= z^b x^c y^d ( e_c {\Gamma^a}_{bd} - e_d {\Gamma^a}_{bc} + {\Gamma^a}_{ec} {\Gamma^e}_{bd} - {\Gamma^a}_{ed} {\Gamma^e}_{bc} - {\Gamma^a}_{be} {c_{cd}}^e ) e_a + x^a y^b {c_{ab}}^d (e_d z^c) e_c - x^a y^b {c_{ab}}^d (e_d z^c) e_c $
$= z^b x^c y^d ( e_c {\Gamma^a}_{bd} - e_d {\Gamma^a}_{bc} + {\Gamma^a}_{ec} {\Gamma^e}_{bd} - {\Gamma^a}_{ed} {\Gamma^e}_{bc} - {\Gamma^a}_{be} {c_{cd}}^e ) e_a $
$= {R^a}_{bcd} z^b x^c y^d e_a$
For ${R^a}_{bcd} = e_c {\Gamma^a}_{bd} - e_d {\Gamma^a}_{bc} + {\Gamma^a}_{ec} {\Gamma^e}_{bd} - {\Gamma^a}_{ed} {\Gamma^e}_{bc} - {\Gamma^a}_{be} {c_{cd}}^e$
In index notation: $([\nabla_c,\nabla_d] - \nabla_{[c,d]}) z^a = {R^a}_{bcd} z^b$

Ricci curvature: $ R_{ab} = {R^c}_{acb} $
Gaussian curvature: $ R = {R^a}_a $
