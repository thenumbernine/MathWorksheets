Let $\Pi = n^\mu \Phi_{,\mu}$
$\Pi = n^t \Phi_{,t} + n^i \Phi_{,i}$
$\Pi = \frac{1}{\alpha} ( \Phi_{,t} - \beta^i \Phi_{,i} )$ (eqn. 5)

Let $\Psi_i = \Phi_{,i}$ (eqn. 6)

Solve $\Pi$ for $\Phi_{,t}$:
$\Pi = \frac{1}{\alpha} ( \Phi_{,t} - \beta^i \Phi_{,i} )$
$\alpha \Pi = \Phi_{,t} - \beta^i \Phi_{,i}$
$\Phi_{,t} = \alpha \Pi + \beta^i \Phi_{,i}$ (eqn. 7)
$\frac{d}{dt} \Phi = \alpha \Pi$

Solve $\Pi_{,i}$ for $\Psi_{i,t}$:
$\Pi = \frac{1}{\alpha} ( \Phi_{,t} - \beta^i \Phi_{,i} )$
$\Pi_{,i} = -\frac{1}{\alpha^2} \alpha_{,i} ( \Phi_{,t} - \beta^j \Phi_{,j} ) + \frac{1}{\alpha} ( \Phi_{,t} - \beta^j \Phi_{,j} )_{,i} $
$\Pi_{,i} = -\frac{1}{\alpha^2} \alpha_{,i} \Phi_{,t} + \frac{1}{\alpha^2} \alpha_{,i} \beta^j \Phi_{,j} + \frac{1}{\alpha} \Phi_{,ti} - \frac{1}{\alpha} {\beta^j}_{,i} \Phi_{,j} - \frac{1}{\alpha} \beta^j \Phi_{,ji} $
substitute $\Phi_{,t}$:
$\Pi_{,i} = -\frac{1}{\alpha^2} \alpha_{,i} (\alpha \Pi + \beta^i \Phi_{,i}) + \frac{1}{\alpha^2} \alpha_{,i} \beta^j \Phi_{,j} + \frac{1}{\alpha} \Phi_{,ti} - \frac{1}{\alpha} {\beta^j}_{,i} \Phi_{,j} - \frac{1}{\alpha} \beta^j \Phi_{,ji} $

substitute $\Phi_{,i} = \Psi_i$
$\alpha \Pi_{,i} = -\alpha_{,i} \Pi + \Psi_{i,t} - {\beta^j}_{,i} \Phi_{,j} - \beta^j \Psi_{i,j} $
solve for $\Psi_{i,t}$
$\Psi_{i,t} = (\alpha \Pi)_{,i} + \beta^j \Psi_{i,j} + {\beta^j}_{,i} \Phi_{,j} $ (eqn. 8)
$\frac{d}{dt} \Psi_i = (\alpha \Pi)_{,i}$
